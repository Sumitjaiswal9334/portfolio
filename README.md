# Sumit Jaiswal Portfolio v2

A responsive, single-page portfolio built with HTML, CSS and JavaScript.

## Update before deployment

Search and replace these dummy values inside `index.html`:

- `https://linkedin.com/in/your-profile`
- `https://github.com/your-username`
- `https://leetcode.com/u/your-profile`
- `your.email@example.com`
- Each project link currently set to `#`

## Run locally

Open `index.html` in a browser, or use VS Code Live Server.

## Deploy

Upload the folder to Netlify. The contact form is configured for Netlify Forms.
