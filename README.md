# Sumit Jaiswal Portfolio

A modern, responsive single-page portfolio website for Sumit Jaiswal, focused on AI/ML, Generative AI, full-stack development, projects, skills, milestones, education, and contact details.

## Overview

This portfolio is built as a static website using HTML, CSS, and JavaScript. It highlights selected AI, machine learning, and full-stack projects with live demo links, GitHub links, project screenshots, resume access, and a Netlify-ready contact form.

## Features

- Responsive single-page layout
- Hero section with profile image, role highlights, resume links, and quick stats
- About section focused on AI and software engineering experience
- Featured project cards with screenshots, tech stack, live demo links, and GitHub links
- Project archive with category filters
- Skills section grouped by technical areas
- Milestones and achievements section
- Education and coursework section
- Contact form configured for Netlify Forms
- Smooth reveal animations and interactive UI behavior
- SEO-friendly meta tags and favicon support

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Netlify Forms



## Run Locally

Because this is a static website, you can run it directly in the browser.

1. Open the project folder.
2. Open `index.html` in your browser.

You can also use VS Code Live Server for a better local development experience.

## Deployment

This project can be deployed on any static hosting platform.

Recommended options:

- Netlify
- Vercel
- GitHub Pages

For Netlify deployment:

1. Upload or connect the `portfolio_v2` folder.
2. Keep the publish directory as the project root.
3. Netlify will automatically detect the contact form because `index.html` includes `data-netlify="true"`.

## Customization

To update the portfolio content, edit:

- `index.html` for text, links, project details, education, and contact information
- `style.css` for layout, colors, spacing, and responsive design
- `script.js` for animations, menu behavior, filters, and interactions
- `assets/` for images, logos, project screenshots, and resume PDF
